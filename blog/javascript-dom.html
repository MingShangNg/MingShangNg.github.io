Javascript , HTML & CSS

JavaScript (JS) is the most popular text-based programming language that allows users to make web pages interactive. HypeText Markup Language (HTML) is the standard markup language for defining the structure of the web content. Cascading Style Sheets is (CSS) a style sheet language used for styling the presentation of web content. These three languages work together to allow users to create web pages and web applications.

Analogy for Javascript, HTML and CSS

Think like our human body. HTML is like a skeleton. It provides our body structure. CSS is like skin and hair. It provides style to our body (eg, different colour of skin, hair). Javascript is like the brain. It allows us to interact.

Control flow and Loops

By default the control flow of Javascript runs the code from the first line on top to the end line on bottom, however we can change the control flow by setting up some statements such as loops. Changing the control flow allows the application to be more interactive and dynamic. Loops are one of the statements that let flow keep running until there is either nothing left to loop over, or if the condition becomes false.

Example of control flow for daily life will be:

  1. Hungry
  2. Cooking
  3. eating

Example of loop for daily life will be:

  1. Hungry
  2. Cooking
  3. Eating
  4. If still hungry
  5. Cook
  6. Eat
  7. Loop again till not hungry

DOM and an example interact with it.

The DOM stands for Document Object Model. It is a programming interface that allows users to interact with the elements from the document. Document is just a webpage such as an HTML document. Objects are the elements in the HTML such as the <head>. The model describes the object layout such as tree branches. All elements in the HTML are nodes and we can use it for manipulation by using (getElementsById, getElementsByTagName, etc).

Arrays and Object

In order to understand arrays and objects. We need to understand variables. In JavaScript, we can use variables to store any data value by using the syntax var, let and const. Objects are mutable data structures in javascript which are used to represent a “Thing” with characteristics. In objects, these characteristics are called properties that consist of a key and a value. let object = { key : ‘value’.} Properties in objects can be accessed by using either dot or bracket notation. Person.age return or person [‘age’].Arrays are variables which can store multiple values by using special syntax let i = [].Every value inside the array is using zero-based indexing so it means the first value in an array has index 0, second value will be index of 1 and so on. We can use this index to access the information from the array.

Function

In Javascript, function is the fundamental build block of code designed to perform a particular task. A function will execute when the user invokes it (calls it). A function definition consists of the function keyword, followed by name of function, a list of parameters to the function () and statements that define the function {}.For example, function name(parameters){}. Functions are very useful because they allow the user to create an interactive application.
