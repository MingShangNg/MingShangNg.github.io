CSS stands for Cascading Style Sheets. It is used to style the web pages design and layout. In order to understand how CSS style the layout, we need to understand the CSS box model. The CSS box model refers to the box wraps around the HTML elements. Every box consists of four areas: content, padding, border and margin.

Content

The content area is the area that shows the content of the element, such as text, an image or video player.

Padding

The padding area is the empty space around the content. The size of the area of padding can be determined by the padding-top, padding-right, padding-bottom and padding left.

Border

The border is the border that goes around the padding and content. There are three type border properties, border-width, border style and border-colour.

Margin

The margin area is empty outside the border. The size of the area of padding can be determined by the margin-top, margin-right, margin-bottom and margin left.
